# Voice Notes App

A simple and interactive Voice Notes App built using HTML, CSS, and JavaScript. The application allows users to record voice notes, convert speech into text, organize notes by category, search notes, and store data using Local Storage.

## Features

### Speech-to-Text
- Record voice notes
- Convert speech into text
- Real-time transcription using Web Speech API

### Notes Management
- Create notes
- Save notes
- Delete notes
- View saved notes

### Categories
- Study
- Work
- Personal
- Ideas

### Search Functionality
- Search notes by content
- Search notes by category
- Instant filtering

### Dashboard
- Total Notes Counter
- Real-time statistics

### Data Persistence
- Save notes using Local Storage
- Automatically restore notes after page refresh

### User Interface
- Clean and responsive design
- Easy-to-use controls
- Beginner-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Web Speech API
- Local Storage API

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Voice-Notes-App.git
```

2. Navigate to the project directory

```bash
cd Voice-Notes-App
```

3. Open `index.html` in your browser.

4. Allow microphone access when prompted.

## How It Works

1. Click **Start Recording**.
2. Speak into your microphone.
3. The speech is converted into text automatically.
4. Select a category.
5. Click **Save Note**.
6. Search notes using the search box.
7. Delete notes when no longer needed.
8. Notes are automatically saved in Local Storage.
